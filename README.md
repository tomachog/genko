Proceso de creación del proyecto

El desarrollo de este proyecto se llevó a cabo mediante un enfoque práctico y progresivo, combinando aprendizaje autodidacta, uso de recursos online y herramientas externas para optimizar tanto el diseño como la funcionalidad del sitio web.

Para la estructura y el diseño visual de la página utilicé librerías encontradas en internet, las cuales me permitieron acelerar el desarrollo y mantener una interfaz ordenada y moderna. En particular, implementé Bootstrap para crear componentes responsivos, destacándose el uso de su carrusel de imágenes, que facilita la visualización dinámica del contenido.

Durante el proceso de estilado, recurrí a tutoriales de YouTube, especialmente de canales enfocados en desarrollo web y diseño frontend como Fazt, Midudev, Soy Dalto y Kevin Powell, que explican buenas prácticas de CSS, organización visual y experiencia de usuario. Estos recursos fueron clave para mejorar la presentación del proyecto y comprender cómo estructurar correctamente los estilos.

En los casos donde encontré limitaciones técnicas o conceptos que no lograba resolver únicamente con herramientas de inteligencia artificial, opté por parchear y adaptar soluciones manualmente, probando distintas alternativas hasta lograr el comportamiento deseado. Esto me permitió reforzar el aprendizaje práctico y entender mejor el funcionamiento del código.

Además, utilicé Gemini en su modo de aprendizaje guiado, lo cual me ayudó a comprender paso a paso ciertos conceptos, en lugar de solo copiar soluciones. Esta herramienta fue utilizada como apoyo educativo para afianzar conocimientos y mejorar la toma de decisiones durante el desarrollo.

Para la creación de imágenes y elementos gráficos del proyecto, empleé Canva, aprovechando sus plantillas y herramientas de diseño para generar recursos visuales acordes a la temática del sitio y mantener una identidad visual coherente.

En conclusión, el proyecto es el resultado de la combinación entre investigación, práctica constante, uso responsable de herramientas externas y aprendizaje guiado, aplicando conceptos vistos durante el curso de desarrollo web y complementándolos con recursos disponibles en la web.
